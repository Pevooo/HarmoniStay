# HarmoniStay
Visual Programming and Software Engineering project

This is an all-in-one hotel management system that offers various features:
1) Staff Management
2) Financial Management
3) Room Management
4) Booking Management
5) Chatbot (powered by OpenAI)
6) Event Management
